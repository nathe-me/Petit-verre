# PETIT VERRE - Official Rules book

Le jeu est simple ; on va tous finir a terre.

## RULES :
1. Le mot "shot" est interdit dans un bar. Celui qui le dit doit payer sa tournee a quiconque l'a entendu.
2. Pieger quelqu'un pour qu'il dise le mot est autorise et meme recommande.
> ### Exemples :
> - "Comment il s'appelle le son de Bob Marley dans lequel il parle de tuer le sherif ?"
> - "Comment on dit tir dans la tete en anglais ?"
> - ...
3. Le but n'est pas de ruiner les autres, mais de rester dans une bonne ambiance. Donc si une personne est insolvable ou que c'est chaud, elle est exclue du jeu.
4. Si on est exclu du jeu, on peut dire le mot mais on ne peut recevoir le petit verre si quelqu'un d'autre le dit.

## EXCEPTIONS
1. En parlant de cafe on peut dire "shot d'expresso" ou "double shot d'expresso"
